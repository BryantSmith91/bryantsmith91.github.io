How to Get Started with Python Programming

Step 1: Install Anaconda

Click this link to go to the download page for "Anaconda", scroll down and, select the installer for your OS.

Step 2:

Run the installer that downloads and go through all of the steps.

Take extra care and read the options when it comes up with the "Advanced Options" as shown to the right. I'd advise following their recomendations and selecting the options as shown to the right which I've also noted below.




Step 3:

Next, you'll go ahead and click "Install" and Anaconda should automatically install with no errors, hopefully, and once installed, you've got Python installed with all of the packages in the Anaconda library. Some of these packages are listed below, please note the full list of packages can be accessed here.

  • Python : General purpose programming language

  • Pandas : High-performance, easy-to-use data structures and data analysis tools.

  • pip : PyPA recommended tool for installing Python packages

  • Jupyter : Jupyter metapackage. Install all the Jupyter components in one go

  • Matplotlib : Publication quality figures in Python

Step 4:

The installer will then provide you with some options to learn more about Anaconda, I'd recommend reading up on these and opening the "Anaconda Navigator" program that was just installed on your computer. But, now you have Python and Anaconda installed, you can start the next steps of learning about Data Science Programming and using Pandas in the next entry of this tutorial series which will be linked to here when the post is ready.
